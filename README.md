# 项目介绍
这是一个电商后台管理系统项目
技术栈：前端：vue2 、vue-router、element_ui、vue/cli、axios
可以拿来当作练手的项目，欢迎star
##
打开git bush输入以下命令/n
git clone https://github.com/weiyouchang/project_01.git/n
将项目下载到本地
## Project setup
```下载依赖
npm install
```

### Compiles and hot-reloads for development
```启动项目
npm run serve
```

### Compiles and minifies for production
```打包发布
npm run build
```
